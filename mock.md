# Cloud Computing Questions and Answers

## Question 1

A deployment model has the following characteristics:

* Easy to set up and use
* Low data security and privacy
* Cheapest cost

### Answer

**Public Cloud** is the deployment model that best matches these characteristics.

**Characteristics:**

* Easy to set up and use
* Lowest upfront cost (pay-as-you-go model)
* Lower data security and privacy compared to private or hybrid cloud
* Generally the cheapest deployment option

## Question 2

__________ is the use of software to handle routine cloud computing tasks, such as provisioning servers, deploying applications, and scaling resources, without manual intervention. For example, instead of manually creating 10 virtual machines, you can write a script to automatically create them based on specific configurations.

### Answer

**Cloud Automation**

Cloud Automation is the use of software, scripts, and tools to handle routine cloud computing tasks such as provisioning servers, deploying applications, configuring infrastructure, and scaling resources without manual intervention.

## Question 3

In which cloud computing model do you only need to manage:

* Applications
* Data
* Runtime
* Middleware
* Operating System

### Answer

**Infrastructure as a Service (IaaS)**

In IaaS, the cloud provider manages the physical infrastructure, including servers, storage, networking, and virtualization, while the customer is responsible for managing the operating system and everything above it.

---

## Question 4

__________ is a managed service that sets up and operates a relational database in the cloud.

### Answer

**Database as a Service (DBaaS)**

DBaaS is a managed cloud service that sets up, operates, maintains, and scales a relational database in the cloud. The cloud provider handles installation, backups, updates, security patches, and maintenance.

Examples:

* Amazon RDS
* Azure SQL Database
* Google Cloud SQL

## Question 5

__________ is a fully managed Platform as a Service (PaaS) that simplifies application deployment and scaling on AWS by automating infrastructure provisioning and management.

### Answer

**AWS Elastic Beanstalk**

AWS Elastic Beanstalk is a fully managed Platform as a Service (PaaS) that simplifies application deployment, management, and scaling. It automatically handles infrastructure provisioning, load balancing, auto scaling, and application health monitoring.

## Question 6

Explain in your own words why we need cloud computing. Do we need to implement cloud computing in all applications?

### Answer

Cloud computing allows organizations and individuals to access computing resources such as servers, storage, databases, and software over the internet without purchasing and maintaining their own physical infrastructure.

### Benefits of Cloud Computing

1. **Cost Savings** – Reduces the need for expensive hardware and data centers.
2. **Scalability** – Resources can be increased or decreased quickly based on demand.
3. **Flexibility and Accessibility** – Applications and data can be accessed from anywhere with an internet connection.
4. **Faster Deployment** – New applications and services can be deployed quickly.
5. **Reliability** – Cloud providers offer backup, disaster recovery, and high availability.
6. **Reduced Maintenance** – Infrastructure management and updates are handled by the cloud provider.

### Do All Applications Need Cloud Computing?

No, not all applications need cloud computing.

Some applications may be better suited for on-premises or hybrid environments because of:

* Strict security, privacy, or regulatory requirements
* Very low latency requirements
* Dependence on local infrastructure
* High migration costs for legacy systems

Therefore, cloud adoption should be based on business requirements, budget, security needs, and performance expectations.

# Desktop Virtualization in Healthcare

## Question 7

Healthcare is one of the industries that can benefit the most from desktop virtualization, particularly in time management, financial cost management, and security management. Using your own words, explain in detail how desktop virtualization helps with TWO (2) of the mentioned management domains in healthcare.

### Answer

### 1. Time Management

Desktop virtualization improves time management by allowing healthcare professionals to access their desktop environment, applications, and patient records from any authorized device. Doctors, nurses, and administrative staff can move between departments and continue their work without needing a specific workstation.

For example, a doctor can access a patient's electronic health record (EHR) from the emergency room, clinic, or ward using the same virtual desktop. Software updates and maintenance are performed centrally, reducing downtime and increasing productivity. This allows healthcare workers to spend more time focusing on patient care and less time dealing with technical issues.

### 2. Security Management

Desktop virtualization enhances security management by storing patient data on centralized servers rather than on local devices. If a laptop, tablet, or workstation is lost or stolen, sensitive patient information remains protected because it is not stored on the device itself.

Healthcare organizations can also implement centralized security controls such as user authentication, access permissions, encryption, and automatic security updates. These measures help protect confidential medical records, reduce the risk of data breaches, and support compliance with healthcare privacy regulations.

### Conclusion

Desktop virtualization provides significant benefits to healthcare organizations by improving time management through faster access to applications and patient records and enhancing security management through centralized data storage and stronger security controls. These improvements help healthcare providers deliver efficient, secure, and high-quality patient care.
